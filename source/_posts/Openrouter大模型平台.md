# OpenRouter 使用
## 为什么使用OpenRouter？
1. Openrouter聚合了多家模型，一站式调用
2. 有很多模型提供商都会在 OpenRouter 提供测试的预发布模型，token免费😀

## 如何充值？
1. 访问[OpenRouter](https://openrouter.ai)
2. 要填写有关的信息（邮政编码地址之类），接着访问[OpenRouter点数](https://openrouter.ai/settings/credits)
![图1](/images/Openrouter大模型平台/image1.png)
3. 点击"Add Credits"，出现一个这样的界面：
![图2](/images/Openrouter大模型平台/image2.png)
4. 在这个界面，你可能找不到国内的支付方式的选项，其实只需打开页面中的“Use One-time payment methods”开关，此时你就会发现你可以使用微信或者支付宝充值点数了

一般的，你需要充值10美元，调用免费模型才不会受到限制。

## 获取API kEY
访问[Openrouter API KEY ](https://openrouter.ai/workspaces/default/keys)，创建API KEY后即可将API KEY用于各类AI服务，比如Zed、Opencode。
